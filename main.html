Example of usage: To add a download item: GET request to '/:botnumber/downloads' with query parameters: jidlist, url, title, caption To remove a jid from a download item: GET request to '/:botnumber/downloads' with query parameters: jid, done=true To get an updated download item: GET request to '/:botnumber/downloads' with query parameters: jid, target=true To get all download items: GET request to '/:botnumber/downloads/all' To add bot details: POST request to '/:botnumber' with body parameters: groupjid, type To get bot details: GET request to '/:botnumber/:groupjid'
